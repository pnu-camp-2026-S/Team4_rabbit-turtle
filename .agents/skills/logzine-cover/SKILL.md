# logzine-cover — 취향 기반 매거진 표지 생성 규칙

MY COVER("이번 주 나의 표지") 기능이 Gemini 이미지 생성(`gemini-2.5-flash-image`)으로
사용자 취향 태그를 **실제 잡지 표지 한 장**으로 만드는 규칙을 정의한다.
구현 위치: `logzine_app/lib/services/cover_art_service.dart` (`_coverPrompt`)

## 레퍼런스 문법 (실제 잡지 표지에서 추출)

| 레퍼런스 | 가져온 규칙 |
|---|---|
| VOGUE | 대형 세리프 제호가 상단 전폭, **피사체가 제호를 살짝 가려도 됨** |
| GOURMET TRAVELLER | 제호 아래 굵은 커버라인 + 피사체(음식)가 화면 중앙 주인공, 따뜻한 톤 |
| EHOUSING | 왼쪽 세로로 흐르는 작은 목차 라인, 상단 구석 발행 정보 |

## 생성 규칙 (프롬프트에 코드화됨)

1. **제호**: "LOGZINE" 대형 세리프 대문자, 상단 전폭. 피사체가 하단을 살짝 겹칠 수 있음
2. **피사체**: 사용자 취향 태그들을 **하나의 자연스러운 장면으로 합성**
   - 예: `축구 + 파스타` → 축구장에 차린 테이블에서 파스타를 먹는 사람
   - 예: `커피 + 카메라` → 필름 카메라와 푸어오버 커피의 정물
3. **커버라인**: 제호 아래 2~4단어 영문 볼드 한 줄 (취향 무드 요약) + 왼쪽 가장자리에 아주 작은 서브라인 2~3개
4. **스타일**: 하이엔드 에디토리얼 사진 — 부드러운 자연광, 따뜻한 뮤트 톤, 필름 그레인, 얕은 심도 (Kinfolk/Gourmet Traveller 무드)
5. **금지**: 바코드·가격·워터마크·오탈자·한글 텍스트 (이미지 모델의 한글 렌더링이 불안정하므로 텍스트는 영문만)

## 안전망 (반드시 유지)

- 생성 실패/쿼터 초과(429) → **타이포그래피 표지 폴백** (`my_cover_page.dart` `_CoverArtwork`)
  — 폴백도 하나의 스타일로 보이도록 조판되어 있음. 제거 금지.
- 캐시: 주 단위 + 취향 해시 (`v2-...`). 규칙을 바꾸면 캐시 키 버전을 올릴 것 (v2→v3)
- 생성 이미지에는 제호가 포함되므로 **Flutter 오버레이를 얹지 않는다**

## 규칙 수정 시

1. `_coverPrompt` 수정 + 캐시 키 버전 업
2. 이 문서 동기화
3. 무료 쿼터 소진에 주의 — 테스트 생성은 하루 수 회로 제한하고, 데모 전날 미리 캐시를 만들어 둘 것
