# design_ref — 디자인 시안 참고 폴더

claude.ai/design에서 만든 디자인을 여기에 넣으면, Claude가 읽어서 Flutter 위젯으로 옮깁니다.

## 넣는 방법
화면별로 HTML/CSS 파일을 아래 이름으로 저장하세요 (있는 것만):

- `signup.html`   — 회원가입
- `interest.html` — 관심사 입력
- `explore.html`  — 매거진 탐색
- `create.html`   — 매거진 생성
- `mypage.html`   — 마이페이지

CSS가 별도 파일이면 같이 넣어주세요. 스크린샷 이미지(`*.png`)도 참고용으로 함께 넣으면 좋습니다.

파일 대신 채팅에 코드를 붙여넣거나 스크린샷을 첨부해도 됩니다.
